# TJ — 삼초경 (Triple Jiao)

> 특정 경선이나 장기가 아닌  
> **전신 자율신경 분배 네트워크**

---

## 핵심 정의

| 항목 | 내용 |
|------|------|
| 위치 | **없음** (전신 분포) |
| 장기 | **없음** |
| 경선 고정 | **불가** |
| 신경학적 실체 | 자율신경총 (Autonomic Plexus) |
| ANS 레벨 | 레벨 2 (신경절 불안정) |

---

## 왜 SBAL이 아닌가

기존 해석: TJ = SBAL (팔 뒤쪽 경선)에 포함

**오류:**
```
TJ = 전신 유동 네트워크
→ 팔 뒤쪽(SBAL)으로 위치 고정 불가
→ SBAL에 포함 시:
   전신 네트워크를 '팔 뒤쪽 경선'으로 격하시키는 이론적 오류
```

**결론:** TJ는 [[Neuro-Fascial ANS Framework]]에서 **독립 분배 레이어**

---

## ANS 시스템에서의 역할

```
횡격막 (ANS 게이트)
    ↓
TJ (전신 분배)
    ↓
SFAL (상지 조절)  ←→  SBAL (상지 출력)
    ↓
SBL / SFL / LL (하지 출력)
```

## TJ 이상 임상 패턴

- 한열왕래 (寒熱往來)
- 전신 불안
- 비정형 통증 (위치 불일정)
- [[상한론 ANS 붕괴 모델|소양병]] 패턴과 대응
