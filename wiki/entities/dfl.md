# DFL — Deep Front Line (심부전면선)

> 지면 반력을 내측 압력으로 변환하여  
> 복압과 척추 안정성을 형성하는 **코어 생성·입력 시스템**

---

## 절대 원칙

> **DFL만이 원인이다. 나머지는 표현이다.**  
> 출력 레이어(SBL·SFL·LL)만 치료하면 반드시 재발한다.

---

## 해부학적 연속선

```
족저근막
    ↓
후경골근 / FDL / FHL
    ↓
내전근군 (Adductors)
    ↓
골반저근 (Pelvic Floor)
    ↓
장요근 (Iliopsoas)
    ↓
횡격막 (Diaphragm) ← ANS 게이트
    ↓
경부 심부굴근 (Longus colli / Longus capitis)
```

---

## 경맥 대응

| 근육 구간 | 경맥 |
|----------|------|
| 내전근 | SP (비경) — 전면 루트 |
| 장요근·골반저 | KI (신경) — 심부 축 |
| 횡격막·측면 | LR (간경) — 측면·회전 루트 |

### 두면부 분지

```
횡격막 이후 3갈래:

SP (비) → 복직근 → 흉근 → 전면 얼굴 (처짐·무거움)
KI (신) → 척추 → 인후 → 깊은 목 (삼킴 문제)
LR (간) → 늑골 → 측면 → 측두부·턱·눈 (두통·턱 긴장)
```

---

## 신경 특성

- 부교감 기반
- 구심성 우세 (정보 입력)
- ANS 레벨 1 (중추)

---

## 핵심 기능

1. 내측 압력 유지
2. 복압 형성
3. 척추 안정
4. 장부 상태의 근막 번역

---

## 임상 단서

| 소견 | 의미 |
|------|------|
| 복직근 긴장 | 교감↑ |
| 복직근 무력 | 비경(DFL) 붕괴 |
| 장요근 단축 | 골반 전방경사 + 요추 과전만 |
| 후경골근 약화 | 편평족 + 내측 압력 소실 |
| 골반저근 경직 | 하초 부교감 조절 이상 |

---

## 횡격막 분기 (ANS 게이트)

```
DFL (압력 시스템)
    ↓ 횡격막
    ↓
LU ← C3-5 체성신경 → 기계적 호흡 (숨이 안됨)
PC ← 미주신경      → ANS·감정 조절 (답답함)
```

> 같은 구조에서 **기계(LU) vs 기능(PC)** 이 갈린다
