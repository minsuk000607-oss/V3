# 경맥 유주·전변·낙맥 — 신경과학 재해석

> 경맥이 이어진 것은 전변되는 순서와 방향을 미리 그려둔 지도다.  
> 낙맥은 그 전변이 일어나는 신경절 간 교차 연결 경로다.

---

## 핵심 명제

```
낙맥 = 자율신경절 간 교차 연결
전변 = 신경절 파급의 단계적 확산
합병증 = 낙맥 경로를 통한 경맥 간 전변
기경8맥 = 자율신경 간선망
```

---

## 낙맥의 신경과학적 실체

### 낙맥 (絡脈) = 신경절 간 교차 연결

기존 해석: 표리 경맥을 연결하는 지맥

**새로운 정의:**
```
주 경맥 = 해당 척수 분절 자율신경절 주 경로
낙맥    = 인접/표리 신경절로의 교차 연결 경로

폐경 낙맥 → 대장경
= T1-T4 교감 신경절 → T6-T12 교차 연결
```

### 임상 예시

| 원발 장기 | 낙맥 경로 | 합병증 | 신경과학 기전 |
|---------|---------|--------|------------|
| 심장 (T1-T4) | 비경 낙맥 | 소화기 합병증 | 심부전 → 위장 울혈 |
| 간 (T6-T10) | 신경 낙맥 | 신장 합병증 | 간경화 → 신부전 |
| 신장 (T10-L2) | 심경 낙맥 | 심장 합병증 | 신부전 → 심부전 |

---

## 경맥 유주 = 전변 순서와 방향의 지도

### 육경 전변 = 신경절 파급의 단계적 확산

```
태양병 (T1-T4 교감↑ 과활성)
↓ 치료 안 되면
교감 탈감작 → 인접 신경절 파급
↓
양명병 (T6-T12 열 팽창)
↓
소양병 (신경절 불안정·교대 진동)
↓
태음병 (부교감 일방 장악)
↓
소음병 (중추 각성 실패)
↓
궐음병 (전체 루프 붕괴)
```

> 경맥이 이어진 순서 = 자율신경계가 붕괴하는 순서

### 합병증 발생의 법칙

```
병이 한 경맥에서 → 낙맥을 통해 → 표리/인접 경맥으로 파급
= 서양 의학에서 장기 합병증이 생기는 이유
= 자율신경절이 인접 신경절로 파급되기 때문
```

## 기경8맥 = 자율신경 간선망

| 기경 | 신경학적 실체 | 역할 |
|------|------------|------|
| 督脈 | 교감신경 척추 간선 | 전신 교감 긴장 조절 |
| 任脈 | 부교감·미주신경 간선 | 전신 부교감 조절 |
| 陽維·陰維 | 교감·부교감 신경절망 | 전변 확산 경로 |
| 帶脈 | T12-L2 횡격막-골반 연결 | DFL-SBL 교차점 |

```
병이 깊어질수록 → 기경8맥 레벨 치료 필요
합병증이 생기면 → 독맥·임맥 먼저
```

## IMS·Gunn = 경근병의 서양 재발견

```
Gunn이 발견한 것          경맥이 말한 것
──────────────────────────────────────
신경근 압박               경근(經筋) 긴장
→ 분절 근육 단축          → 해당 경맥 근육 반응
→ 신경인성 염증           → 사기(邪氣) 결체
→ 분절 침자로 해결        → 경근 따라 침자

언어만 다를 뿐, 같은 실체를 봤다
```

## 임상 적용 5단계 프로토콜

```
1단계 — 맥진 (자율신경 단계 판별)
맥부긴 → 태양·교감↑
맥현   → 소양·신경절 불안정
맥미세 → 소음·탈진

2단계 — 장부 증상 유무
있음 → 경맥병 (자율신경 루프)
없음 → 경근병 (근막 레벨)

3단계 — 경맥병 치법
교감↑ → 독맥 사법 + 배수혈 + 황금·황련
부교감↑ → 반하·생강 + 임맥 조율
탈진   → 독맥·임맥 보법 + 인삼·부자·건강

4단계 — 근막 정리
경맥병 치료 후 잔존 → 근막경선 침 사법

5단계 — MPS·IMS
근막 해소 후 잔존 → Travell MPS → Gunn IMS
```

### 치료 우선순위

```
처방 (수용체·자율신경 루프 재조정)
↓
독맥·임맥·배수혈 (자율신경 간선)
↓
경혈 원위·근위 (루프 재가동)
↓
근막경선 (체성 원심성 정리)
↓
MPS·IMS (잔존 국소)
```

