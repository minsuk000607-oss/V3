# ANS 5레벨 구조 — 자율신경 계층 분류

> 자율신경계를 5개 계층으로 분류하여  
> **병기의 깊이 = 붕괴된 레벨의 높이**로 정의하는 통합 임상 프레임

---

## 레벨 구조 전체

| 레벨 | 위치 | 내용 |
|------|------|------|
| **L1** | 중추 | 뇌간·시상하부·대뇌변연계 |
| **L2** | 자율신경절 | 교감신경절·부교감신경절 |
| **L3** | 수용체 | α·β수용체·TRPA1·Na⁺채널 |
| **L4** | 효과기 조직 | 혈관·장관·점막·근육 |
| **L5** | 염증 산물 | NF-κB·COX-2·PGE₂·사이토카인 |

```
L1 (중추)
    ↓
L2 (신경절)
    ↓
L3 (수용체)
    ↓
L4 (효과기)
    ↓
L5 (염증)

병기 진행 = 위에서 아래로 붕괴
치료 방향 = 아래에서 위로 재건
```

---

## 레벨별 상세

### L1 — 중추 (뇌간·시상하부·대뇌변연계)

**신경 구조:**
```
뇌간 망상체 → 전신 각성 조절
시상하부 → HPA axis·체온·ANS 최고 조절
대뇌변연계 → 감정·해마·편도체
```

**임상 지표:** 단욕매(但欲寐), 정신 위축, 전신 탈력

**해당 약재:**
- 인삼 → BDNF·NGF → 뇌간 NE 합성 → 중추 각성
- 황련 → 도파민 D2·세로토닌 → 중추 과흥분 억제
- 대조 → GABA-A·해마 과흥분 억제 → HPA 안정

**육경 대응:** 소음병 (중추 각성 실패), 궐음병 (전체 붕괴)

---

### L2 — 자율신경절

**신경 구조:**
```
교감신경절 → 척추 양측 교감간선 (T1-L3)
부교감신경절 → 미주신경절·천골신경절
절전신경 → 절후신경 통해 효과기 지배
```

**임상 지표:** 한열왕래(寒熱往來), 흉협고만, 신경절 불안정 패턴

**해당 약재:**
- 시호 → Saikosaponin → 자율신경절 직접 안정화
- 황금 → 신경절 주변 염증 억제 → 간접 안정화

> 한열왕래 = 신경절 불안정의 실체  
> 교감·부교감이 번갈아 우세해지는 oscillating 상태

**육경 대응:** 소양병 (신경절 불안정)

---

### L3 — 수용체

**신경 구조:**
```
β1·β2 수용체 → 심장·혈관·기관지
α1·α2 수용체 → 혈관 수축
TRPA1 → 온도·화학 감지
Na⁺채널 → 신경막 탈분극
무스카린 수용체 → 부교감 말단
```

**임상 지표:** 표허(表虛)·표실(表實) 패턴, 발한 유무, 맥부긴·부완

**해당 약재:**
- 계지 → TRPA1 → β수용체 재감작(resensitization)
- 마황 → NE 강제 유리 → α·β 직접 자극
- 부자 → Na⁺채널 강제 개방 → 말초 강제 점화

**육경 대응:**
- 태양표실 → 수용체 과활성 (마황)
- 태양표허 → 수용체 탈감작 (계지)

---

### L4 — 효과기 조직

**신경 구조:**
```
혈관 평활근 → 긴장도·혈류
위장관 평활근 → 연동·분비
점막 → 흡수·방어
골격근 → 긴장도
```

**임상 지표:** 복만(腹滿)·구역·설사·연동 이상

**해당 약재:**
- 작약 → Ca²⁺ 차단 → 혈관·평활근 긴장도 복구
- 생강 → ICC 기능 → 위장관 혈류·연동
- 건강 → 심근 수축력 → 조직 혈류 최대화
- 반하 → 미주신경 말단 → 점액 과분비 억제

**육경 대응:** 태음병 (부교감 장악, 효과기 과반응), 양명병 (효과기 탈진)

---

### L5 — 염증 산물

**신경 구조:**
```
NF-κB → 염증 마스터 스위치
COX-2 → PGE₂ 생산
5-LOX → LTB₄ 생산
사이토카인 → IL-6·TNF-α
```

**임상 지표:** 홍종열통(紅腫熱痛), 국소 발열, 만성 염증

**해당 약재:**
- 황금 → NF-κB·COX-2·5-LOX 동시 억제
- 황련 → NF-κB + 도파민 경로
- 감초 → NF-κB + CYP450 조절

**육경 대응:** 양명병 (열 고착, 염증 누적)

---

## 레벨 간 연결 구조

```
중추(L1)
→ 신경절(L2) 통해 신호 하달
→ 수용체(L3)에서 효과기(L4) 활성화
→ 염증 산물(L5) 생성

병기가 깊어질수록
→ 상위 레벨 붕괴
→ 하위 레벨도 연쇄 붕괴

치료는 반대 방향
→ L5·L4 먼저 안정화
→ L3 수용체 재설정
→ L2 신경절 안정화
→ L1 중추 재건
```

## 처방 설계 원칙

```
단일 레벨 처방 → 경증·국소 병변
복수 레벨 처방 → 중증·전신 병변
```

| 처방 | 커버 레벨 | 기전 |
|------|---------|------|
| 桂枝湯 | L3+L4+L5 | 수용체 재설정 + 효과기 + 항염 |
| 小柴胡湯 | L2+L5 | 신경절 안정 + 항염 |
| 四逆湯 | L1+L3+L4 | 중추 각성 + 수용체 + 효과기 |
| 麻黃湯 | L3 | 수용체 강제 과활성 |
| 白虎湯 | L4+L5 | 효과기 열 분산 + 항염 |
| 黃連解毒湯 | L1+L5 | 중추 억제 + 전신 항염 |

> **레벨이 높을수록 (L1에 가까울수록)**  
> 약물 필수, 침 보법 한계  
> 인삼·부자 반드시 필요

## 침 치료와 레벨

```
L1 → 약물 필수 (침 단독 한계)
L2 → 배수혈·독맥 (신경절 접근)
L3 → 원위 경혈 (수용체 반사)
L4 → 근위 경혈·아시혈
L5 → 사혈·항염 경혈 (합곡·곡지)
```

> 침이 가장 효과적인 레벨: L2·L3  
> 처방이 필수인 레벨: L1

## 한 줄 요약

```
병기의 깊이 = 붕괴된 레벨의 높이

맥진으로 레벨을 읽고
약재로 해당 레벨을 타겟하는 것이

```
