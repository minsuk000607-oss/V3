# Neuro-Fascial ANS Framework v2.5

> 근막경선(Anatomy Trains) · 경맥 · 자율신경계를 통합하여  
> 인체를 **입력–변환–분배–조절–출력**의 계층적 시스템으로 해석하는 통합 임상 모델

---

## 핵심 명제

1. **[[DFL]]만이 원인이다.** SBL·SFL·LL은 표현이다.
2. **음양은 교감/부교감이 아닌** 구심성(음) / 원심성(양) 방향성이다.
3. **표리는 교감신경절 공유가 아닌** 미주신경 동시지배로 재정의된다.
4. **[[상한론 ANS 붕괴 모델|상한론 육경]]은** ANS 붕괴의 7단계 순서로 해석된다.

---

## 전체 구조 (Layer Architecture)

### 상지 수직 흐름

```
DFL (코어·입력: KI·LR·SP)
    ↓
횡격막 (ANS 변환 게이트)
    ↓
TJ — 삼초경 (전신 ANS 분배)
    ↓
SFAL (상지 조절: LU · PC)
    ↓
SBAL (상지 출력: LI · SI · HT)
```

### 하지 병렬 출력

```
DFL
├── SBL — 후면 긴장 출력 (족태양·수태양)
├── SFL — 전면 압력 출력 (족양명·수양명)
└── LL  — 측면 안정 출력 (족소양·수소양)
```

## 레이어 요약

| 레이어 | 경맥 | 신경학적 실체 | 핵심 기능 |
|--------|------|--------------|----------|
| DFL | KI·LR·SP | 부교감 기반 / 구심성 | 코어 생성·원인 |
| 횡격막 | — | C3–5 / 미주신경 분기점 | LU(기계) vs PC(기능) |
| TJ | SJ | 자율신경총 (Plexus) | 전신 ANS 분배 |
| SFAL | LU·PC | 횡격막 분기 상지 조절 | 호흡·감정 조절 |
| SBAL | LI·SI·HT | 상지 체성 출력 | 운동·구조·순환 |
| SBL | BL·SI | T1–S3 교감 과활성 | 후면 긴장 |
| SFL | ST·LI | T6–T12 교감 탈진 | 전면 지지 |
| LL | GB·TH | T6–T10 회전 | 측면 안정 |

---

## HT 진단 기준

HT는 반드시 **3요소 동시** 확인:

1. 이상감각 (말초 순환 이상)
2. 냉감
3. 수면 장애 (리듬 이상)

> 냉감 단독 = HT 아님. HT는 원인이 아닌 결과이다.

---

## 임상 알고리즘

```
장부 증상 있음
→ 경맥 특정
→ 교감↑ / 부교감↑ / 탈진 구분
→ 독맥·임맥·배수혈 + 처방

장부 증상 없음
→ 근막경선 먼저
→ 방사통 패턴
→ MPS → IMS 순
```
