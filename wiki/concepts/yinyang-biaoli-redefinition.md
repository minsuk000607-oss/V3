# 음양·표리 재정의 — 신경과학적 번역

---

## 음양 재정의

### 기존 해석 (폐기)

> 음 = 부교감신경  
> 양 = 교감신경

**문제:** 교감/부교감은 기능적 분리이고,  
실제 임상에서는 같은 장기에 교감·부교감이 동시 작용한다.

---

### 새로운 정의

| 개념 | 신경학적 실체 | 방향 |
|------|-------------|------|
| **陰 (음)** | 구심성 신호 (Afferent) | 말초 → 중추 |
| **陽 (양)** | 원심성 신호 (Efferent) | 중추 → 말초 |

```
음 = 정보가 들어오는 방향 (감각·수용)
양 = 명령이 나가는 방향 (운동·분비)
```

### 임상 적용

```
음허 = 구심성 입력 부족
    → 피드백 감소 → 조절 실패

양허 = 원심성 출력 부족
    → 효과기 반응 저하

음양 불균형 = 입력-출력 비율 이상
```

---

## 표리 재정의

### 기존 해석 (폐기)

> 표리 = 교감신경절 공유

**문제:** 폐(T1-T4)와 대장(T6-T12)은  
교감신경절을 공유하지 않는다.  
그런데 임상적으로 명백히 표리 관계다.

---

### 새로운 정의

> **표리 = 미주신경(CN X) 동시지배**

```
폐수 T3 ≠ 대장수 T11
→ 교감신경절 다름

그러나
폐 ← 미주신경 → 대장
→ 동일한 미주신경이 양쪽 지배

= 표리 관계 성립
```

### 6쌍 표리의 신경학적 근거

| 표 | 리 | 공유 신경 | 통로 |
|----|-----|----------|------|
| 폐 (LU) | 대장 (LI) | 미주신경 | T1-T4 · T6-T12 |
| 위 (ST) | 비 (SP) | 미주신경 복강지 | T6-T12 |
| 심 (HT) | 소장 (SI) | 미주신경 심장지 | T1-T4 |
| 방광 (BL) | 신 (KI) | 골반내장신경 | T1-S3 · S2-S4 |
| 심포 (PC) | 삼초 (TJ) | 미주신경 | T1-T5 |
| 담 (GB) | 간 (LR) | 미주신경 복강지 | T6-T10 |

---

## 통합 해석

```
경맥 =
자율신경 구심성으로 받고 (음)
체성신경 원심성으로 내보내는 (양)
쌍방향 반사 루프의 체표 지도

근막경선 = 경맥의 체표 레벨만
경맥 = 근막경선 + 자율신경 루프 + 장부
```

---

## 임상 함의

표리 치료의 원리:
```
대장 문제
→ 미주신경 공유 경로로
→ 폐 기능에 영향
→ 폐 계열 혈위 병용 가능

= 표리 취혈의 신경학적 근거
```
