# 상한론 육경 — ANS 붕괴 7단계 모델

> 장중경의 육경 체계는 **자율신경계의 단계적 붕괴 과정**에 대응하는  
> 임상 경험 기반 신경과학 모델이다.

---

## 핵심 전제

```
12경맥 = 6개 통로 × 교감·부교감
표리 쌍 = 하나의 신경절 통로를 교감·부교감으로 나눠 표현한 것

폐·대장 = T1-T4·T6-T12 하나의 통로
위·비   = T6-T12 하나의 통로
심·소장 = T1-T4 심장 통로
방광·신 = T1-S3·S2-S4 통로
심포·삼초 = T1-T5 자율신경 통로
담·간   = T6-T10 통로
```

---

## 7단계 진행 모델

### 1단계 — 태양표실 (太陽表實)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 교감신경 **급성 과활성** |
| 수용체 | NE → α·β 아드레날린 수용체 과자극 |
| 주증상 | 발열·오한·무한·체통·부긴맥 |
| 처방 | **마황탕** (에페드린: NE 강제 유리) |
| 블록 | C3–T1 출력 (경항·체표 긴장) |

### 2단계 — 태양표허 (太陽表虛)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 교감 **탈감작** 시작 |
| 수용체 | β수용체 내재화(internalization) |
| 주증상 | 발열·오한·유한·맥완 |
| 처방 | **계지탕** (신남알데히드: 수용체 재설정) |
| 블록 | C3–T1 (탈감작 시작) |

### 3단계 — 양명경증 (陽明經證)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 교감 탈진 + **혈관 내 열 팽창** |
| 수용체 | 체온조절 없는 혈관 확장 |
| 주증상 | 대열·대한·대갈·맥홍대 (4대증) |
| 처방 | **백호탕** (CaSO₄: 혈관 내 열 분산) |
| 블록 | T6–T12 압축 |

### 4단계 — 양명부증 (陽明腑證)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 장관 자율신경 **기능 실패** + 열 고착 |
| 수용체 | 내압 상승 |
| 주증상 | 조시·복만통·불대변·맥침실 |
| 처방 | **승기탕** (안트라퀴논: 연동 강제 유도) |
| 블록 | L2–L4 전방 엔진 |

### 5단계 — 소양 (少陽)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 자율신경절 **불안정** (교감·부교감 교대 진동) |
| 수용체 | 신경절 oscillating |
| 주증상 | 한열왕래·흉협고만·목현·이청 |
| 처방 | **소시호탕** (Saikosaponin: 신경절 안정화) |
| 블록 | T6–T12 (흉협 담위 불화) |

### 6단계 — 태음 (太陰)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 부교감 **일방 장악** |
| 수용체 | 무스카린 수용체 과활성 |
| 주증상 | 복만·시자리·복통·설사 |
| 처방 | **이중탕** (Atractylenolide+Shogaol: L4 회복) |
| 블록 | S2–S4 지지 붕괴 |

### 7단계 — 소음→궐음 (少陰→厥陰)

| 항목 | 내용 |
|------|------|
| ANS 상태 | 중추 각성 실패 → **수용체 소실** |
| 수용체 | Na⁺채널 임계 저하 |
| 주증상 | 맥미세·단욕매·사지궐냉→열궐 |
| 처방 | **사역탕→오매환** (Na⁺채널 강제 개방) |
| 블록 | T12–L2 축 붕괴 → 전체 통로 |

## 맥상 → ANS 상태 대응

| 맥상 | ANS 상태 | 기전 |
|------|----------|------|
| 浮緊 (부긴) | 교감 급성 과활성 | 말초 혈관 수축 |
| 浮緩 (부완) | 교감 탈감작 시작 | β수용체 내재화 |
| 洪大 (홍대) | 혈관 확장·열 팽창 | 체온조절 실패 |
| 沈實 (침실) | 장관 내압 상승 | 자율신경 기능 실패 |
| 弦 (현) | 신경절 불안정 | 교감·부교감 oscillating |
| 微細 (미세) | 중추 ANS 기능 실패 | 심박출량 임계 감소 |

## 블록별 육경 매핑

| 블록 | 해당 육경 | 임상 패턴 |
|------|----------|----------|
| C3–T1 | 태양표증 / 소양발열 / 궐음탁음 | 경항·체표 긴장 출력 |
| T6–T12 | 소양흉협 / 양명열성 | 흉복부 압축·집중 |
| T12–L2 | 소음한화(신양허) / 소음열화(신음허) | 축 붕괴 |
| L2–L4 | 양명리실 / 태음중초한습 | 복강 충만 |
| L4–S2 | 어혈하초 / 기울사역 | 사말 경맥 폐색 |
| S2–S4 | 태음한증하리 / 소음고섭실패 | 지지 붕괴 |

## 폐·대장 통로 붕괴 순서

```
양명부증 (대장 열 고착·조시)
↓ 해소 안 되면
소양 (신경절 불안정·한열왕래)
↓
태음 (부교감 장악·해수·흉민 시작)
↓
소음 (호흡근 무력·숨참)
↓
궐음 (전체 통로 붕괴)
```

> 대장에서 시작해서 표리인 폐까지 끌어내리는  
> **하나의 통로 순차 붕괴**
