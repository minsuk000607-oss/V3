# BL1선·BL2선 이중 레이어 이론 (v2)

> 척추 레벨별 자율신경 구조의 차이가  
> 배수혈·BL2선 분포 패턴을 결정한다.  
> "공백"이 아니라 **자율신경 조절 기전 자체가 전환되는 구간**이다.

---

## 핵심 재정의

### 기존 해석 (v1)
```
L3-S1 = 자율신경 공백
→ 그래서 BL2선 없음
→ 전면 접근 우선
```

### 수정 해석 (v2)
```
척추 전체를 절전 유무로 재분류:

절전 접근 가능 구간: T1-L2, S2-S4
신경절/신경총 레벨만 존재: C1-C7, L3-S1

→ 배수혈이 있는 구간 = 절전 레벨 접근
→ 배수혈 없거나 기전이 다른 구간 = 신경절/신경총 레벨
```

---

## 구간별 자율신경 구조

### C1-C7 구간

```
교감 절전: 없음 (T1에서 시작)
교감 신경절:
  상경부 교감신경절(SCG): C2-C3
  중경부 교감신경절: C6
  성상신경절: C7-T1
  (절전은 T1-T5에서 올라와 시냅스)

부교감:
  뇌신경(CN III·VII·IX·X)
  미주신경(CN X) = 경부 전체 주행
  연수(medulla) = C1 직상방 = ANS 최고 중추

접근 방식:
  독맥(GV) → CNS·교감 간선
  풍지(GB20) → 미주신경 + SCG 교차점
  천주(BL10) → C2-C3 SCG 절후 레벨
  천돌(CV22) → 미주신경 직접
```

### T1-L2 구간 ★ 절전 접근 가능

```
교감 절전: 척수 T1-L2 측각(lateral horn)에서 기원
교감 신경절: 척추 양측 교감간선 신경절
→ 절전이 신경절에 시냅스

BL1선(배수혈): 교감신경절 직접 접점
BL2선: 근막 직접 접근 레이어

→ 이 구간만 배수혈로 절전 직접 조절 가능
```

### L3-S1 구간 ★ 기전 전환

```
교감 절전: 없음 (L2에서 끊김)
교감 신경절:
  L3-S1 교감신경절은 존재
  단, 절전이 T12-L2에서 하행해 시냅스한 절후만
  → 절후섬유가 체성신경(마미총, 회백교통지)에 통합

부교감: 없음 (S2에서 시작)

자율신경 조절 기전 전환:
  절전 레벨 조절 불가
  → 자율신경총(신경절 집합체)으로 전환
  → 하복신경총(L4-S1), 골반신경총이 전면에 위치

BL2선 없는 이유:
  절후만 있고 체성신경에 통합
  → 근막 직접 접근도 자율신경 직접 조절도
     배수혈/BL2선 방식이 효과 제한적

전면 접근 우선 이유:
  신경총(교감+부교감+체성 통합)이 전면 복강에 분포
  → 전면 자침 = 신경총 직접 접근
  → 임맥 뜸 = 신경총 온열 활성
```

### S2-S4 구간 ★ 부교감 절전

```
부교감 절전: 척수 천골 S2-S4 측각에서 기원
→ 골반내장신경(pelvic splanchnic n.)
→ 골반신경총 → 하행결장·직장·방광·자궁·생식기

髎穴(上次中下料): 천골공 직접 → 부교감 절전 직접 접근
= T1-L2 배수혈과 동일한 절전 접근 방식
```

## 전체 접근 체계 재정리

| 구간 | 절전 여부 | 주 접근 방식 | 대표 혈위 |
|------|---------|------------|---------|
| C1-C7 | 교감 절전 없음 | 신경절 + CNS | 풍지(GB20)·천주(BL10)·독맥 |
| T1-L2 | 교감 절전 있음 | 배수혈(절전 직접) + BL2선 | BL11~BL23·BL41~BL52 |
| L3-S1 | 절전 없음 | 신경총(전면) + 임맥 | 관원·기해·오추·하복신경총 |
| S2-S4 | 부교감 절전 있음 | 髎穴(절전 직접) | 상·차·중·하료 |

## 임상 함의

### 경추부 접근 (C1-C7)
```
배수혈 없음 = 절전 접근 불가
→ 신경절 레벨 접근 필요

독맥 IMS: CNS + 교감 간선
풍지(GB20): 미주신경 + SCG 동시
성상신경절 주변 자침: 두경부 교감 직접
```

### T1-L2 접근
```
배수혈 = 절전 직접 접근
BL2선 = 근막 레이어 추가
→ 병용 시 ANS + 근막 동시
```

### L3-S1 접근
```
후면 배수혈 = 절후 레벨만 (효과 제한)
→ 전면 신경총 접근 우선

전면 임맥 뜸: 복강신경총 온열 직접 활성
오추(GB27): 장요근 기시부 + 하복신경총 근방
관원·중극: 하복신경총 직접
```

### S2-S4 접근
```
髎穴 = 부교감 절전 직접
전면 중극·곡골 보조
→ 골반 장기 부교감 회복
```

## QL 통증 재해석 (업그레이드)

```
QL 상부(T12-L2): 절전 레벨 존재
→ BL2선 있음 = 근막 + 절전 양쪽 접근
→ 근막·내장 양쪽 원인 가능

QL 하부(L3-L4): 절전 없음
→ BL2선 없음 = 근막 직접 접근 제한
→ 절후만 + 신경총 레벨
→ 내장-체표 반사가 주 기전
→ 전면 접근 우선
```

---

## 노인성 신허 케이스 (임상 관찰)

```
처방: 오추(GB27) + 동일 레벨 전면 혈위
이유:
  L4-L5 = L3-S1 공백 구간
  → 배수혈 효과 제한 (절후만)
  → 전면 하복신경총 직접 접근
  → 오추 = 장요근 기시부 (DFL) + 하복신경총 근방
  → 신장 주변 근막 직접 해소 + 신경총 자극

= 절전 공백 구간에서 전면 신경총 접근의
  임상 검증 사례
```
