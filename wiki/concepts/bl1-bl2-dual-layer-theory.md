# BL1선·BL2선 이중 레이어 이론

> BL1선(후정중선 1.5촌)과 BL2선(후정중선 3촌)의 분포 패턴은  
> 각 수평에서의 **통증 기원(내장 vs 근막)**을 반영하는  
> 경험적 설계 체계일 수 있다.

---

## 발단 — 요방형근 분포와 BL2선의 불일치

요방형근(QL)은 T12부터 L4 수평까지 분포한다.

그런데 BL2선은 L2 志室에서 끊긴다.

```
T12 胃倉(BL50) ← QL 있음
L1  肓門(BL51) ← QL 있음
L2  志室(BL52) ← QL 있음 (belly 구간)
──────────────────── BL2선 끝
L3  기해유(BL24) ← BL1선만 존재
L4  대장유(BL25) ← BL1선만 존재
```

단순한 해부학적 한계(장골릉 차단)로 볼 수도 있으나,  
더 근본적인 이유가 있을 수 있다.

---

## 핵심 가설

> L3-L4 수평의 외측 통증(QL 하부 섬유 긴장)은  
> **근육 기원성이 아니라 내장 기원성**이다.
>
> 따라서 근막 직접 접근(BL2선)이 아니라  
> 배수혈을 통한 장부 ANS 조절(BL1선)이 설계된 것이다.

### 신경해부학적 근거

```
L3-L4 분절 = 대장·소장·자궁·방광 내장신경 레벨

내장 ANS 이상
→ 내장-체표 반사(viscerosomatic reflex)
→ 해당 분절 체표·근육 긴장
→ QL 하부 긴장·통증으로 표현

= "QL 하부 통증"처럼 보이지만
  실제 원인은 내장 ANS 이상
```

---

## 이중 레이어 분류 체계

### BL1선 (배수혈, 후정중선 1.5촌)

```
역할: 장부 ANS 조절 레이어
기전: 교감신경절 접점 → 해당 내장 ANS 직접 조절
치료 대상: 내장 기원성 통증 / 내장 기능 이상
```

### BL2선 (BL2선, 후정중선 3촌)

```
역할: 근막 직접 접근 레이어
기전: 척주기립근·QL 등 심층 근막 직접 자극
치료 대상: 근막 기원성 통증 / 근육 긴장
```

### 두 선이 겹치는 수평

```
BL1선 + BL2선 양쪽 있음
→ 해당 수평 통증이 근막 + 내장 양쪽에서 기인
→ 양 레이어 모두 필요

T1-L2: 양쪽 다 있음
→ 이 구간은 근막 긴장과 내장 ANS 이상이 동시 기여
```

### BL2선이 없는 수평

```
BL1선만 있음
→ 해당 수평 통증의 주요 기원이 내장
→ 배수혈 ANS 조절만으로 충분

L3-S4: BL1선만
→ 이 구간 통증은 내장 ANS 조절이 우선
```

---

## 수평별 분포 패턴 전체

| 수평 | BL1선 | BL2선 | 주요 내장 | 해석 |
|------|-------|-------|---------|------|
| T1-T2 | ✓ | ✓ | 폐·심 | 근막+내장 양쪽 |
| T3-T7 | ✓ | ✓ | 폐·심·심포·격 | 근막+내장 양쪽 |
| T8-T12 | ✓ | ✓ | 간·담·비·위 | 근막+내장 양쪽 |
| L1-L2 | ✓ | ✓ | 삼초·신 | 근막+내장 양쪽 |
| **L3** | ✓ | **✗** | 대장(하부) | **내장 주도** |
| **L4** | ✓ | **✗** | 대장·소장 | **내장 주도** |
| **L5-S4** | ✓ | **✗** | 소장·방광·골반 | **내장 주도** |

## QL로 본 임상 함의

```
QL 상부 통증 (T12-L2):
→ 근막 + 내장 양쪽 가능
→ BL2선 자침 (근막 직접) + BL1선 배수혈 (ANS)
→ 근막병인지 내장병인지 감별 필요

QL 하부 통증 (L3-L4):
→ 내장 기원성 우선 의심
→ 대장유(BL25) 배수혈로 ANS 조절
→ 대장 기능 정상화 → QL 긴장 2차 해소

임상 알고리즘:
L3-L4 수평 통증
→ 장부 증상 함께 있으면? 내장 기원성 확정
→ 배수혈 우선
→ 잔존 긴장만 IMS/아시혈
```

## 기존 이론과의 연결

### [[배수혈 이론]]과의 접점

배수혈이 자율신경절 접점 + 내장-체표 반사 집결점이라면,  
BL2선이 없는 수평에서는 배수혈 하나로  
**내장 ANS 조절과 체표 긴장 해소를 동시에 처리**할 수 있다.

### [[Neuro-Fascial ANS Framework]]과의 접점

```
BL1선 = 자율신경 반사 레이어 (경맥 레이어)
BL2선 = 근막 장력 레이어 (경근 레이어)

두 레이어가 겹치는 구간 = 배수혈과 BL2선 혈위 동시 존재
두 레이어가 분리되는 구간 = 내장 기원성이 주도하는 수평
```

### [[경맥·경근 분리 이론]]과의 접점

경맥(자율신경 레이어)과 경근(근막 장력 레이어)이  
별개 체계임을 BL1선·BL2선 분포 패턴이  
해부학적으로 지지하는 근거가 된다.

---

## 검증 가능한 예측

이 가설이 맞다면:

1. **L3-L4 수평 외측 통증 환자에서**  
   대장·소장 증상 동반률이 상부 QL 통증 환자보다 유의하게 높아야 함

2. **대장유(BL25) 자침만으로**  
   L3-L4 외측 통증이 해소되는 빈도가  
   아시혈 직접 자침보다 높아야 함

3. **L3-L4 QL 긴장의 표면 근전도 패턴이**  
   T12-L2 QL 긴장과 다른 반응 특성을 보여야 함

---

## Research Gap

- BL2선 분포 종료 수평(L2)과 내장-체표 반사 범위의 상관관계 연구 없음
- BL1선 단독 vs BL1선+BL2선 병용 치료 효과 비교 연구 없음
- QL 하부 섬유 긴장의 내장 기원성 비율 미확인
