# IMS·MPS 통합 임상 레퍼런스

> Gunn IMS + Travell MPS + Anatomy Trains을  
> 척수분절 신경해부학 축으로 통합한 즉시적용 임상 레퍼런스

---

## 핵심 개념 비교

| | Gunn IMS | Travell MPS |
|--|----------|-------------|
| 이론 기반 | 신경병증성 통증 / 탈신경 과민 | 근막 통증 유발점 |
| 치료 타겟 | 척수분절 기반 근육군 | 유발점 (TP) 직접 |
| 침 자입 | 분절 전체 | TP 국소 |
| 효과 지표 | LTR (국소 연축반응) | LTR + 연관통 소실 |
| 경혈 대응 | 화타협척혈 / 방광경 배수혈 | 아시혈 |

---

## Gunn IMS 이론

### 핵심: 탈신경 과민 (Denervation Supersensitivity)

```
신경근 압박 (Radiculopathy)
    ↓
해당 분절 근육 탈신경
    ↓
아드레날린 수용체 과민
    ↓
근육 단축·경직 (shortened muscle)
    ↓
통증·기능 제한
```

### 치료 원리

```
분절 침 자입 → LTR 유발
→ 근육 이완 → 신장 반사 차단
→ 혈류 회복 → 신경 재생
```

## 척수분절 임상 매핑

### 경추부 (C3–T1)

| 분절 | 근육 | 임상 |
|------|------|------|
| C3–4 | 승모근 상부, 흉쇄유돌근 | 경항통, 두통 |
| C5 | 삼각근, 극상근 | 견관절 외전 제한 |
| C6 | 이두근, 요측수근굴근 | 엄지·2지 방사통 |
| C7 | 삼두근, 척측수근굴근 | 3지 방사통, 테니스엘보 |
| C8–T1 | 소지구근, 골간근 | 4·5지 이상감각 |

### 요천추부 (L1–S3)

| 분절 | 근육 | 임상 |
|------|------|------|
| L1–2 | 장요근 | 서혜부통, 대퇴 전면 |
| L3–4 | 대퇴사두근 | 슬관절 전면, 내측 |
| L4–5 | 전경골근, 장지신근 | 족배·1지 방사통 |
| L5–S1 | 비복근, 가자미근 | 종아리·5지 방사통 |
| S2–4 | 골반저근 | 회음부·항문 방사통 |

## MPS 유발점 핵심 체인

### 경추–견갑 체인

```
승모근 상부 TP
→ 후두·측두부 연관통
→ 흉쇄유돌근 위성 TP
→ 안면·측두 연관통 확장
```

### 요부–하지 체인

```
장요근 TP
→ 요부·서혜부·대퇴전면 연관통
→ 대퇴직근 위성 TP
→ 슬개골 하방 연관통
```

## 임상 결정 알고리즘

```
환자 주소증
    ↓
장부 증상 있음?
├── Yes → [[Neuro-Fascial ANS Framework]] 적용
│         경맥 특정 → 배수혈 + 처방
└── No  → 근막경선 먼저
          ↓
          방사통 패턴 확인
          ├── 분절 패턴 → IMS 우선
          └── 유발점 뚜렷 → MPS 우선
```

## 경혈 대응표

| IMS/MPS 구조 | 한의학 대응 |
|-------------|-----------|
| 화타협척혈 | 독맥 방광경 1선 |
| 근육 TP (아시혈) | 아시혈 |
| 척수분절 배수혈 | 방광경 배수혈 |
| 신경절 자극점 | 교감신경절 = 배수혈 위치 |

---

## 치료 우선순위

```
1. 처방 (수용체 레벨)
2. 독맥·임맥·배수혈 (신경절 레벨)
3. 경혈 (경맥 레벨)
4. 근막경선 (근막 레벨)
5. MPS → IMS (국소 레벨)
```
